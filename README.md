# prod-stories-HW4
## Требования
* Версия python: 3.8.8
* Версия pip: 21.0.1
* Версия conda: 4.10.3
## Инструкция по запуску
1. Создайте оболочку
<pre>$ conda env create -f environment.yml</pre> 
2. Установите необходимые зависимости
<pre>$ pip3 install -r requirements.txt</pre>
3. Запустите обучение
<pre>$ python3 ppo_train.py</pre>
